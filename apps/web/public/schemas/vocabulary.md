# BharatVerify Schema Vocabulary

This document defines the vocabulary terms used in the `IndianWorkforceCredential` schema.

## Namespace

**Prefix**: `bv-vocab`  
**URI**: `https://bharatverify.io/vocab#`

---

## Terms

### referenceId
- **URI**: `bv-vocab:referenceId`
- **Type**: `xsd:string`
- **Description**: A unique reference identifier for the credential, generated by BharatVerify.
- **Format**: `BV-{timestamp}-{random}`
- **Example**: `BV-1707235200000-A3B2C1`

### aadhaarHash
- **URI**: `bv-vocab:aadhaarHash`
- **Type**: `xsd:string`
- **Description**: SHA-256 hash of the holder's Aadhaar number. The raw Aadhaar is never stored to preserve privacy.
- **Privacy Note**: This field enables identity verification without exposing the actual Aadhaar number.

### fullName
- **URI**: `bv-vocab:fullName`
- **Type**: `xsd:string`
- **Description**: The full legal name of the credential holder.

### dateOfBirth
- **URI**: `bv-vocab:dateOfBirth`
- **Type**: `xsd:date`
- **Description**: Date of birth of the credential holder in ISO 8601 format.
- **ZK-Proof Use**: Can be used to prove age thresholds (e.g., "over 18") without revealing exact date.

### skillSet
- **URI**: `bv-vocab:skillSet`
- **Type**: `xsd:string`
- **Description**: Primary skill set, trade, or professional qualification.
- **Examples**: "Full Stack Development", "Electrician", "Plumber", "Nursing"

### isGraduated
- **URI**: `bv-vocab:isGraduated`
- **Type**: `xsd:boolean`
- **Description**: Boolean indicating whether the holder has completed a graduation-level education.
- **ZK-Proof Use**: Can prove graduation status without revealing institution or degree details.

### cibilScore
- **URI**: `bv-vocab:cibilScore`
- **Type**: `xsd:integer`
- **Description**: CIBIL credit score of the holder. Range: 300-900, or -1 if not applicable.
- **ZK-Proof Use**: Can prove score thresholds (e.g., "score > 700") without revealing exact score.

### institutionName
- **URI**: `bv-vocab:institutionName`
- **Type**: `xsd:string`
- **Description**: Name of the educational institution that issued the credential.
- **Examples**: "Delhi University", "IIT Bombay", "ITI Chandigarh"

### degreeTitle
- **URI**: `bv-vocab:degreeTitle`
- **Type**: `xsd:string`
- **Description**: Official title of the degree or certificate.
- **Examples**: "B.Tech in Computer Science", "Diploma in Electrical Engineering"

### completionYear
- **URI**: `bv-vocab:completionYear`
- **Type**: `xsd:integer`
- **Description**: Year when the degree/certification was completed.

### grade
- **URI**: `bv-vocab:grade`
- **Type**: `xsd:string`
- **Description**: Final grade, CGPA, or classification achieved.
- **Examples**: "8.5 CGPA", "First Class with Distinction", "A+"

---

## Selective Disclosure Strategy

The schema is designed to support **selective disclosure** via Zero-Knowledge Proofs. Key strategies:

1. **Financial Privacy**: `cibilScore` can be proven with range proofs without revealing exact value.
2. **Age Verification**: `dateOfBirth` can prove age thresholds.
3. **Qualification Proof**: `isGraduated` + `skillSet` can prove qualifications without revealing institution.
4. **Identity Binding**: `aadhaarHash` binds to national ID without exposing it.

## Example Usage

```json
{
  "@context": [
    "https://www.w3.org/2018/credentials/v1",
    "https://bharatverify.io/schemas/v1"
  ],
  "type": ["VerifiableCredential", "IndianWorkforceCredential"],
  "credentialSubject": {
    "id": "did:polygonid:polygon:amoy:2q...",
    "referenceId": "BV-1707235200000-A3B2C1",
    "aadhaarHash": "a3f2b1c4d5e6...",
    "fullName": "Rahul Sharma",
    "dateOfBirth": "1995-03-15",
    "skillSet": "Full Stack Development",
    "isGraduated": true,
    "cibilScore": 750,
    "institutionName": "Delhi University",
    "degreeTitle": "B.Tech in Computer Science",
    "completionYear": 2017,
    "grade": "8.2 CGPA"
  }
}
```
